# Job Application Tracker

A full-stack MERN application to track job applications during the internship/job search process.

##  Live Demo

- **Frontend**: https://job-tracker-inky-eta.vercel.app/
- **Backend API**: https://job-tracker-backend-c4li.onrender.com

##  Features

- User Authentication with JWT
- Dashboard with real-time statistics
- Add, edit, and delete job applications
- Status tracking (Applied → Interview → Offer/Rejected)
- Store company details, salary, job URL, and notes
- Responsive design

##  Tech Stack

**Frontend:** React.js, Axios, CSS3  
**Backend:** Node.js, Express.js, MongoDB, JWT, bcryptjs  

##  Installation

### Backend Setup
```bash
cd backend
npm install
# Create .env file with MONGODB_URI, JWT_SECRET, PORT
npm run dev
```

### Frontend Setup
```bash
cd frontend
npm install
npm start
```
